Privacy Policy

Personal data is acquired on this website only to the technically required extent. In no case, will the data collected be sold or disclosed to third parties for any reason. The following information provides you with an overview of the data the website collects.
Log files: While using this tool, the website collects the data your browser transfers to the server. This includes usually information about the type and version of your browser, your operation system, the referrer URL (the page from where you came from before), the host name of the server (IP address) as well as a time stamp and the server request. Usually, it is not possible to relate your visit to a particular person. We do not compare the data with other data we have, neither do we collate/merge different data sets. Data is used only for statistical purposes and is deleted on a regular basis.

This website uses Piwik, an Open Source software for statistical evaluation and analysis of page visits. Piwik uses cookies. The cookie created by Piwik contains information about your use of the website and are saved on our servers in Germany. The IP address is, after its processing, directly anonymized. You can avoid the creation of cookies via your browser settings. However, we advise you that you might not be able to use all functions of the website. Furthermore, you have the option to decline any data process by Piwik for the future. Simply disable the check box below this text. By doing so, Piwik creates a Piwik deactivation cookie in your browser. Please note that this cookie will be deleted, if you delete all cookies in your browser.

Here you can decide if this tool is allowed to create a web analysis cookie in your browser, which helps us to evaluate and analyse different data of your page visit. If you are against it, please follow the link to create a Piwik deactivation cookie in your browser.
